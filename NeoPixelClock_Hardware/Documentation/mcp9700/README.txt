Title   : MCP9700A explanation

****************************** BRIEF DESCRIPTION ******************************
The MCP9700 is a sensor with Low-Power Linear Active Thermistor ICs comprise 
a family of analogue temperature sensors that convert temperature to analogue 
voltage.
Range: 			-40C to +125C
Accuracy: 		+-4C(max) from 0C to +70C
ADC:	 		10.0mV/C
Operating range:	2.3V to 5.5V
Operating current:	6uA

******************************* PERIPHERALS USED *******************************
Port: GPIPB, Pin0, AN
ADC:  ADC1 channel 8
